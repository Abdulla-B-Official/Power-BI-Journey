#  Healthcare Analysis Dashboard | Power BI

An interactive **Power BI Dashboard** designed to analyze **healthcare performance, patient insights, treatment trends, and operational metrics** using business intelligence and data visualization techniques.

##  Project Objective

The goal of this project is to analyze and monitor:

* Patient Demographics & Insights
* Healthcare KPI Tracking
* Treatment & Diagnosis Trends
* Hospital Operational Performance
* Resource Utilization Analysis
* Data-Driven Healthcare Decision Support

This dashboard helps healthcare organizations make **data-driven decisions** by transforming healthcare data into actionable insights.

##  Key KPIs

* **Total Patients**
* **Treatment Analysis**
* **Patient Demographics**
* **Healthcare Performance Metrics**
* **Resource Utilization**
* **Operational Efficiency**

### Business Questions

* How are patient admissions changing over time?
* What are the most common treatments and diagnoses?
* Which patient groups require the most attention?
* How efficiently are healthcare resources being utilized?
* What trends can be identified to improve healthcare services?

##  Project Process

1. **Data Collection** – Imported healthcare datasets into Power BI
2. **Data Cleaning** – Processed and transformed data using Power Query
3. **Data Modeling** – Built relationships between datasets
4. **KPI Creation** – Developed DAX measures for healthcare analysis
5. **Dashboard Development** – Created interactive reports and visualizations
