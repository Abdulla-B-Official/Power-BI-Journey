#  EV Vehicle Analysis Dashboard 

An interactive **Power BI dashboard** built to analyze **electric vehicle adoption, market trends, vehicle performance, and EV growth insights** using business intelligence and data visualization techniques.

##  Project Objective

The goal of this project is to analyze and monitor:

* Electric Vehicle Growth Trends
* EV Market Distribution
* Vehicle Type Analysis
* Manufacturer Performance
* Registration & Adoption Patterns
* Data-Driven EV Insights

This dashboard helps stakeholders understand **EV market trends and adoption patterns** to support informed business and policy decisions.

##  Key KPIs

* **Total EV Registrations**
* **Vehicle Type Distribution**
* **Manufacturer Analysis**
* **EV Growth Trends**
* **Market Insights**
* **Regional Performance Tracking**

### Business Questions

* How has EV adoption changed over time?
* Which manufacturers dominate the EV market?
* Which EV vehicle types are most popular?
* What trends can be identified in EV growth?
* Which regions show higher EV adoption?

##  Project Process

1. **Data Collection** – Imported EV datasets into Power BI
2. **Data Cleaning** – Processed and transformed data using Power Query
3. **Data Modeling** – Built relationships between datasets
4. **KPI Creation** – Developed DAX measures for EV analysis
5. **Dashboard Development** – Created interactive reports and visualizations

This **EV Vehicle Analysis Dashboard** provides a comprehensive overview of **electric vehicle trends, adoption, and market performance** through **interactive visualizations and KPI analysis**, enabling better decision-making using **Power BI**.
