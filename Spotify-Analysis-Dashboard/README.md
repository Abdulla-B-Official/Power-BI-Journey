#  Spotify Streaming Analysis Dashboard | Power BI

An interactive **Power BI Dashboard** designed to analyze **Spotify streaming behavior, listening patterns, artist performance, album popularity, and track engagement metrics** using business intelligence and data visualization techniques.

##  Project Objective

The goal of this project is to analyze and monitor:

* Music Streaming Trends
* Artist & Album Performance
* Track Popularity Analysis
* User Listening Behavior
* Platform Usage Insights
* Listening Time & Engagement Metrics

This dashboard helps uncover **streaming patterns and listener preferences** by transforming Spotify listening history into meaningful and actionable insights.

##  Key KPIs

* **Total Albums Played**
* **Total Artists Played**
* **Total Tracks Played**
* **Average Listening Time**
* **Track Frequency Analysis**
* **Platform Usage Distribution**

### Business Questions

* Which artists and albums are played the most?
* How has listening activity changed over time?
* Which tracks generate the highest engagement?
* What are the most frequently used streaming platforms?
* How do listening patterns vary across days and hours?
* What is the relationship between listening time and track frequency?
* Which albums and artists contribute most to overall listening activity?

##  Project Process

### 1. Data Collection

* Imported Spotify streaming history datasets into Power BI.
* Gathered information related to tracks, albums, artists, listening duration, and streaming platforms.

### 2. Data Cleaning & Transformation

* Processed raw streaming data using Power Query.
* Removed duplicates and missing values.
* Standardized date, time, and categorical fields.
* Created additional columns for analysis.

### 3. Data Modeling

* Established relationships between tracks, artists, albums, and listening metrics.
* Optimized the data model for efficient dashboard performance.

### 4. KPI Creation

Created DAX measures for:

* Total Albums Played
* Total Artists Played
* Total Tracks Played
* Average Listening Time
* Track Frequency
* Platform-Based Streaming Analysis
* Year-over-Year Listening Comparison

### 5. Dashboard Development

Built interactive visualizations including:

* KPI Cards
* Streaming Trend Analysis
* Artist Performance Dashboard
* Album Performance Dashboard
* Listening Heatmaps
* Scatter Plot Analysis
* Platform Usage Filters
* Detailed Drill-Through Tables

##  Key Insights

* The majority of listening activity is concentrated among a small group of favorite artists and albums.
* Streaming behavior varies significantly across different platforms and devices.
* Certain time periods and days show higher listening engagement.
* Track frequency and listening duration reveal user preference patterns.
* Popular artists consistently contribute to a large share of total streams.
* Album-level analysis highlights long-term listening preferences.

##  Conclusion

This **Spotify Streaming Analysis Dashboard** provides a comprehensive view of music consumption behavior through **interactive visualizations, streaming analytics, artist performance tracking, and listening pattern analysis**. The dashboard enables users to better understand their music preferences and explore meaningful insights from Spotify streaming data using **Power BI**.
